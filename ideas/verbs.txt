ADMIN STUFF
-----------
broadcast
    like shout, except overrides all blocking
server
    gets running stats from the server
anchor, unanchor
    anchor something so that it cannot leave its location
clean
    clean out inventory of something (destroy all objects)
exits
    lists all exits in detail.
godmode
    god mode on/off (can't take damage)
invis, vis
    makes you invisible to all players of lower level under all circumstances
quell, peace  (unquell, unpeace)
    stop all fights in your environment   (resume fights)
reset
    reset an object to initial state
resurrect
    return a dead person or creature to the land of the living
return
    send someone back to their last location after a teleport
wizlist
    shows list of all known active wizard users
roll
    roll a die (XdY)
zap
    deal a massive amount of damage to a creature
    

PLAYER STUFF
------------

shout
    shout a message to all players in the mud
yell
    shout a message to everyone in your current location and nearby locations
block, unblock
    block messages from someone
equipment
    show the items you are wearing
flee
    flee from combat in a random direction
kills
    shows a list of your victories
pvp
    toggle pvp
prompt
    customize your prompt
quests
    show a list of the quests you have completed
save
    save your character to disk
skills
    the skills you have
spells
    the spells you know
attack, stop attack
    start/stop combat
cast
    cast a spell on a creature or an object
learn
    learn an ability, skill or spell from someone or something
news
    replay login news
stealth
    toggle stealth, if you possess that skill. (too low stamina -> too tired.)
teach
    teach an ability, skill or spell to another player


ITEM ACTIONS
------------
            
activate, deactivate
turn on, turn off
switch on, switch off
    activate an object, deactivate it again

attach, apply, install, use X on Y
    use something on something else
uninstall, unattach, remove
    break stuff apart again (remove X = remove weapon/armour, remove X from Y = unattach stuff)
    
capture, catch
    capture creature with an object (catch mouse with/in a trap)
free, release, liberate
    set a captured creature free again (free it from a trap)
    
drink
    drink something
eat
    eat something
    
empty
    remove all items from a container
    remove substance from a vessel
fill
    fill a vessel with a substance
 
light, burn
    light a light source.  light thing with other: if it needs ignition
extinguish, douse
    extinguish a light source
    
listen, hear
    listen for sounds

move, shove, swivel, shift, manipulate, turn, rotate
    move an object around (or shove it) -> manipulate something

lockpick
    open a lock with force or with a tool

read
    read something

wear, unwear, remove
    wear/remove armor

wield, unwield, remove
    wield/remove weapon

show
    show an object to someone else
